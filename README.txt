A simple and responsive image slider gallery for WordPress using Slick Carousel.

== Description ==

Image Slider Gallery is a lightweight and responsive WordPress plugin that allows you to create beautiful image sliders with ease. It uses the popular Slick Carousel library for smooth transitions and touch support.

== Features ==

* Responsive design that works on all devices
* Touch support for mobile devices
* Easy to use interface in WordPress admin
* Multiple slider options (autoplay, navigation, etc.)
* Shortcode support for easy embedding
* Lightweight and fast loading

== Installation ==

1. Upload the `image-slider-gallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Image Slider' in the admin menu to create and manage your sliders
4. Use the shortcode `[image_slider_gallery]` to display the slider on any page or post

== Frequently Asked Questions ==

= How do I add images to the slider? =

1. Go to 'Image Slider' in the WordPress admin menu
2. Click 'Add Images' button
3. Select or upload images from the media library
4. Click 'Add to Slider' button
5. Save your changes

= How do I change the slider settings? =

You can adjust the slider settings in the 'Image Slider' admin page. Available options include:
- Autoplay
- Navigation arrows
- Dots navigation
- Transition effects

= How do I add the slider to my pages? =

Use the shortcode `[image_slider_gallery]` in any post, page, or widget. You can also use the following attributes:

* `autoplay` - Enable/disable autoplay (true/false)
* `interval` - Autoplay interval in milliseconds (default: 5000)
* `arrows` - Show/hide navigation arrows (true/false)
* `dots` - Show/hide navigation dots (true/false)

Example: `[image_slider_gallery autoplay="true" interval="3000"]`
